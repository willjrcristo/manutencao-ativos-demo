Otimize a Gestão de Ativos da sua Empresa

Uma solução completa para cadastrar, monitorar e gerenciar a manutenção de seus equipamentos, evitando paradas inesperadas e aumentando a eficiência operacional.

Conheça o Projeto

Arquitetura e Implementação do Projeto

O projeto foi desenvolvido como uma Single-Page Application (SPA), utilizando tecnologias web modernas para criar uma experiência de usuário fluida e reativa, sem a necessidade de recarregar a página durante o uso.

Tecnologias Utilizadas:

  • HTML5: Para a estruturação semântica do conteúdo.
  • Tailwind CSS: Um framework CSS utility-first para a criação rápida de interfaces modernas e responsivas, garantindo uma ótima visualização em desktops e dispositivos móveis.
  • JavaScript (ES6+): Utilizado para toda a lógica da aplicação, incluindo manipulação do DOM, gerenciamento de estado em memória (simulando um banco de dados), e interatividade do usuário.
  • Phosphor Icons: Biblioteca de ícones para uma interface mais intuitiva e agradável.

A escolha dessa stack se deu pela sua simplicidade, rapidez de desenvolvimento e pela facilidade em criar uma demonstração funcional e visualmente atraente em um único arquivo, ideal para prototipagem e apresentação de projetos.

Telas e Funcionalidades

Telas de Cadastro (CRUD)

O sistema possui módulos de cadastro completos para as entidades principais. Cada módulo permite as operações de Criar, Ler, Atualizar e Deletar (CRUD), acessadas através de uma interface modal que centraliza a entrada de dados.

Tela de Consulta/Pesquisa

Para facilitar a localização de registros, todas as telas de listagem contam com um campo de filtro dinâmico. A pesquisa é realizada em tempo real na coluna principal de cada tabela (ex: "Descrição" para Ativos, "Serviço" para Programações), atualizando os resultados instantaneamente.

Tela de Relatório (Dashboard)

A tela inicial funciona como um dashboard, apresentando os Indicadores Chave de Performance (KPIs) mais importantes para uma visão gerencial rápida. Os KPIs exibidos são: Total de Ativos, Total de Manutenções Programadas e Total de Ordens de Serviço.

Funcionalidades Adicionais (Diferenciais)

  • Interface Responsiva: A barra lateral se retrai automaticamente em telas menores para maximizar o espaço útil, exibindo tooltips com os nomes das seções ao passar o mouse sobre os ícones.
  • Confirmação de Exclusão: Para evitar a perda acidental de dados, o sistema solicita uma confirmação do usuário antes de remover qualquer registro.
  • Componente de Modal Genérico: A lógica para abrir, fechar e popular as janelas de formulário foi centralizada, tornando o código mais limpo e reutilizável.

Resultados e Considerações Finais

Resultados Alcançados

O projeto resultou em uma aplicação web funcional que atende com sucesso aos requisitos propostos. A demonstração implementada prova a viabilidade do conceito, oferecendo uma interface intuitiva para o gerenciamento completo do ciclo de manutenção de ativos, desde o cadastro do equipamento até a execução dos serviços.

Desafios e Soluções

Um dos principais desafios foi garantir uma experiência de usuário coesa e reativa sem um backend ou banco de dados real. Isso foi superado através da simulação do banco de dados com objetos JavaScript e da implementação de uma lógica de renderização que atualiza a interface dinamicamente após cada ação do usuário. Outro desafio foi criar uma interface responsiva e funcional, solucionado com o uso estratégico das classes utilitárias do Tailwind CSS.

Sugestões de Melhorias Futuras

  • Integração com Backend e Banco de Dados: Substituir a simulação em memória por uma API RESTful e um banco de dados relacional (como PostgreSQL) ou não-relacional (como MongoDB) para persistência de dados.
  • Autenticação de Usuários: Implementar um sistema de login com diferentes níveis de permissão (ex: Administrador, Gerente, Técnico).
  • Relatórios Avançados: Criar uma seção de relatórios com gráficos interativos (usando bibliotecas como Chart.js ou D3.js) para analisar custos, tempo de parada e performance dos ativos.
  • Sistema de Notificações: Enviar alertas por e-mail ou na própria interface sobre manutenções programadas que estão próximas do vencimento.
